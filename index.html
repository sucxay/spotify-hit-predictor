POST /predict

Single track inference. Send a JSON body with 13 audio features. Returns prediction, label, hit/not-hit probabilities, and latency.

POST /predict/batch

Array of up to 500 track objects. Same schema as single, wrapped in a list. All predictions returned in one response.

POST /predict/csv

Upload a .csv file with the required feature columns. Extra columns are silently ignored. Max 500 rows.

GET /health

Returns model load status, model path, and API health. Use this to verify the server is live before sending predictions.

GET /features

Lists all required feature names, their types, and valid ranges. Handy for building your own client or validating CSV headers.

Run locally

Clone, install deps, train the model with python train.py, then serve with uvicorn main:app --reload.
